# Tech Stack

> 相关索引：[[README]]

| 技术 / 工具                    | 职责                            |
| ------------------------------ | ------------------------------- |
| **pnpm workspaces**            | Monorepo 包管理                 |
| **Turborepo**                  | 构建任务编排与增量缓存          |
| **TypeScript strict**          | 全栈类型系统                    |
| **Vite**                       | 客户端构建与开发服务器          |
| **React 18**                   | 游戏界面（HUD、地图、日志）     |
| **Express**                    | HTTP 服务端框架                 |
| **tsx**                        | TypeScript 直接运行与开发热重载 |
| **HTTP REST**                  | 玩家操作请求（客户端 → 服务端） |
| **SSE**                        | 游戏状态推送（服务端 → 客户端） |
| **DeepSeek API**               | 怪物 Agent AI 推理              |
| **Vitest**                     | 单元测试与集成测试框架          |
| **Supertest**                  | HTTP 路由集成测试               |
| **Prettier**                   | 代码格式化                      |
| **ESLint + typescript-eslint** | 静态分析                        |
| **lint-staged**                | 提交前仅检查暂存文件            |
| **pino**                       | 结构化日志                      |
